8C ABSTRACT: THIS SUBROUTINE OPENS AND INITIALIZES A NEW BUFR MESSAGE

13C JDATE. IF THERE IS ALREADY A BUFR MESSAGE OPEN WITHIN MEMORY FOR

15C BEFORE OPENING THE NEW ONE.
