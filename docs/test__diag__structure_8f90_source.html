7!> Aim: Unit test for create_diagnostic_structure and end_diagnostic_structure
8!> Tests:
9!> * diag_id should be greater than the number of domains
10!> * The number of variables in the diangostic domain should be equal to the
11!> sum of the number of variables in every domain
12!> * The variable and dimension names should stay the same if there is 1 domain
13!> in the state
14!> * The variable and dimension names should be appended with _d0# where # is the
15!> domain number if there are multiple domains.
16!> * If you end the diagnostic structure the number of variables should be 0.
17!> * If you end the diagnostic structure the size should be 0.
18!> * Test that you can call create and destroy multiple times without error.
19!> * Check that you can add_domains until you read max_num_domains then the code
20!> should error out (so you can't overwrite the diagnostic domain).
21
