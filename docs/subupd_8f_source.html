5C SUBPROGRAM: SUBUPD
6C PRGMMR: WOOLLEN ORG: NP20 DATE: 2003-11-04
7C
8C ABSTRACT: THIS SUBROUTINE PACKS UP THE CURRENT SUBSET WITHIN MEMORY
9C (ARRAY IBAY IN COMMON BLOCK /BITBUF/) AND THEN TRIES TO ADD IT TO
10C THE BUFR MESSAGE THAT IS CURRENTLY OPEN WITHIN MEMORY FOR LUNIT
11C (ARRAY MBAY IN COMMON BlOCK /BITBUF/). IF THE SUBSET WILL NOT FIT
12C INTO THE CURRENTLY OPEN MESSAGE, THEN THAT MESSAGE IS FLUSHED TO
13C LUNIT AND A NEW ONE IS CREATED IN ORDER TO HOLD THE CURRENT SUBSET.
14C IF THE SUBSET IS LARGER THAN AN EMPTY MESSAGE, THE SUBSET IS
15C DISCARDED ANDA DIAGNOSTIC IS PRINTED. THIS SUBROUTINE IS IDENTICAL
16C TO BUFR ARCHIVE LIBRARY SUBROUTINE MSGUPD EXCEPT SUBUPD DOES NOT PAD
17C THE PACKED SUBSET.
18C
19C PROGRAM HISTORY LOG:
20C 2003-11-04 J. WOOLLEN -- ORIGINAL AUTHOR (WAS IN VERIFICATION
21C VERSION)
22C 2003-11-04 D. KEYSER -- UNIFIED/PORTABLE FOR WRF; ADDED
23C DOCUMENTATION

26C DART $Id$
