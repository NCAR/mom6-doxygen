program obs_diag
 The programs defines a series of epochs (periods of time) and geographic regions and accumulates statistics for these epochs and regions. All 'possible' observation types are treated separately. The results are written to a netCDF file. If the rank histogram is requested (and if the data is available), only the PRIOR rank is calculated.
