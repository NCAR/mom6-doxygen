5c subprogram: getlens

8c abstract: this SUBROUTINE unpacks and returns all of the individual
9c section lengths of the bufr message stored in array mbay, up to a
10c specified point. it will work on any message encoded using bufr
11c edition 2, 3 or 4. the start of the bufr message(i.e. the string
12c "BUFR") must be aligned on the first four bytes of mbay.
13c
14c PROGRAM history log:
15c 2005-11-29 j. ator -- original author
16c dart $id$
17c
18c usage: CALL getlens (mbay, ll, len0, len1, len2, len3, len4, len5)
19c input argument list:
20c mbay - INTEGER: *-WORD PACKED BINARY ARRAY CONTAINING
21c bufr message
22c ll - INTEGER: NUMBER OF LAST SECTION FOR WHICH THE LENGTH
23c is to be unpacked. in other words, setting ll = n
24c means to unpack the lengths of sections 0 through n
25c(i.e. len0, len1,...,len(n)). any section lengths
26c that are not unpacked are returned with a default
27c VALUE of -1.
28c
29c output argument list:
30c len0 - length of section 0 (= -1 IF not unpacked)
31c len1 - length of section 1 (= -1 IF not unpacked)
32c len2 - length of section 2 (= -1 IF not unpacked)
33c len3 - length of section 3 (= -1 IF not unpacked)
34c len4 - length of section 4 (= -1 IF not unpacked)
35c len5 - length of section 5 (= -1 IF not unpacked)
36c
37c remarks:
38c this routine calls: iupb iupbs01
39c this routine is called by: cktaba cnved4 dumpbf mesgbc
40c msgwrt rdbfdx upds3 writlc
41c normally not called by application

43c
44c attributes:
45c language: fortran 77

47c
