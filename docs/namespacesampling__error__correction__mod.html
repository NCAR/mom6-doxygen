Detailed Description

Correct covariances for fixed ensemble sizes. Ref: Anderson, J., 2012: Localization and Sampling Error Correction in Ensemble Kalman Filter Data Assimilation. Mon. Wea. Rev., 140, 2359-2371, doi: 10.1175/MWR-D-11-00013.1.

query the needed table sizes, and read in the values for any given ensemble size. the two arrays of values returned are the true_correl_mean and alpha.
