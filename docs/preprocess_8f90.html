program preprocess

Define the observation types and obs/state quantities understood by DART.

Takes a list of observation type module path names and physical quantity module path names and creates two output files that are compiled into the rest of the DART executables. They define the quantities and observation types that are recognized and processed in any way.

The output module for observation types can contain multiple fragments of standard F90 that may be required to implement forward observation operators for DART. The sections are retrieved from the files by this program and inserted into the appropriate blanks in the DEFAULT_obs_def_mod.F90 and DEFAULT_obs_kind_mod.F90 templates.

The final obs_def_mod.f90 and obs_kind_mod.f90 that are created contain the default code plus all the code required from the selected observation type modules. Preprocess also inserts the required identifiers and strings for the corresponding observation quantities.
