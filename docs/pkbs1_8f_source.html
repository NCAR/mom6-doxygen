8c abstract: this SUBROUTINE stores a specified integer value into a
9c specified location within section 1 of the bufr message stored in
10c array mbay, overwriting the VALUE previously stored at that
11c location. it will work on any message encoded using bufr edition
12c 2, 3 or 4. the start of the bufr message(i.e. the string "BUFR")
13c must be aligned on the first four bytes of mbay, and the location
14c within which to store the VALUE is specified via the mnemonic
15c s1mnem, as explained in further detail below.
