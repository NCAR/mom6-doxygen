subroutine, public obs_def_utilities_mod::track_status (ens_size, val_istatus, val_data, istatus, return_now)
 track_status can be used to keep track of the status of each ensemble member during multiple calls to model_interpolate for a given obs_def. It assumes that you are starting with istatus(:) = 0 If debugging, return_now is only set to true if all istatuses are non-zero If not debugging, return_now is set to true if any istatues are non-zero and any remaining zero istatues are set to 1.
