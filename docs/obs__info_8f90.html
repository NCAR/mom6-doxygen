program obs_info

print out information about observation sequence file(s). summarizes obs types, times, counts.

If 'csv_style_output = .true.' and 'output_file="summary.csv"', the resulting comma-separated-values file can be trivially read into matlab with:

X = readtable('summary.csv')

The first line of the output file describes the columns.
