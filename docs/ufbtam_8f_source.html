5C SUBPROGRAM: UFBTAM
6C PRGMMR: WOOLLEN ORG: NP20 DATE: 1994-01-06
7C
8C ABSTRACT: THIS SUBROUTINE READS SPECIFIED VALUES INTO INTERNAL ARRAYS
9C FROM ALL DATA SUBSETS IN BUFR MESSAGES STORED IN INTERNAL MEMORY.
10C THE DATA VALUES CORRESPOND TO MNEMONICS, NORMALLY WHERE THERE IS NO
11C REPLICATION (THERE CAN BE REGULAR OR DELAYED REPLICATION, BUT THIS
12C SUBROUTINE WILL ONLY READ THE FIRST OCCURRENCE OF THE MNEMONIC IN
13C EACH SUBSET). UFBTAM PROVIDES A MECHANISM WHEREBY A USER CAN DO A
14C QUICK SCAN OF THE RANGE OF VALUES CORRESPONDING TO ONE OR MORE
15C MNEMNONICS AMONGST ALL DATA SUBSETS FOR A GROUP OF BUFR MESSAGES
16C STORED IN INTERNAL MEMORY, NO OTHER BUFR ARCHIVE LIBRARY ROUTINES
17C HAVE TO BE CALLED. THIS SUBROUTINE IS SIMILAR TO BUFR ARCHIVE
18C LIBRARY SUBROUTINE UFBTAB EXCEPT UFBTAB READS SUBSETS FROM MESSAGES
19C IN A PHYSICAL BUFR FILE. UFBTAM CURRENTLY CANNOT READ DATA FROM
20C COMPRESSED BUFR MESSAGES.
