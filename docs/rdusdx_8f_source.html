4C
5C SUBPROGRAM: RDUSDX
6C PRGMMR: WOOLLEN ORG: NP20 DATE: 1994-01-06
7C
8C ABSTRACT: THIS SUBROUTINE READS AND PARSES A FILE CONTAINING A USER-
9C SUPPLIED BUFR DICTIONARY TABLE IN CHARACTER FORMAT, AND THEN STORES
10C THIS INFORMATION INTO INTERNAL ARRAYS IN COMMON BLOCK /TABABD/ (SEE
11C REMARKS FOR CONTENTS OF INTERNAL ARRAYS). THIS SUBROUTINE PERFORMS
12C A FUNCTION SIMILAR TO BUFR ARCHIVE LIBRARY SUBROUTINE RDBFDX,
13C EXECPT THAT RDBFDX READS THE BUFR TABLE DIRECTLY FROM MESSAGES AT
14C BEGINNING OF AN INPUT BUFR FILE.
15C
