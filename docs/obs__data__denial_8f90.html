program obs_data_denial
 open an obs_seq file and randomly change the error variance of N of each obs type to a huge value before copying obs over to the output file. the value of N is namelist-settable. this effectively removes the impact of those observations on the assimilation. it is intended to help implement a data-denial experiment where you assimilate all obs (the original obs_seq file) and then with some obs disabled (the output obs_seq from this program) and compare the results. the default is to remove N of each obs type but the types of obs is also namelist-settable.
