5C SUBPROGRAM: NUMTAB
6C PRGMMR: WOOLLEN ORG: NP20 DATE: 1994-01-06
7C
8C ABSTRACT: THIS SUBROUTINE FIRST SEARCHES FOR AN INTEGER IDN,
9C CONTAINING THE BIT-WISE REPRESENTATION OF A DESCRIPTOR (FXY) VALUE,
10C WITHIN THE INTERNAL BUFR REPLICATION ARRAYS IN COMMON BLOCK
11C /REPTAB/ TO SEE IF IDN IS A REPLICATION DESCRIPTOR OR A REPLICATION
12C FACTOR DESCRIPTOR. IF THIS SEARCH IS UNSUCCESSFUL, IT SEACHES FOR
13C IDN WITHIN THE INTERNAL BUFR TABLE D AND B ARRAYS IN COMMON BLOCK
14C /TABABD/ TO SEE IF IDN IS A TABLE D OR TABLE B DESCRIPTOR. IF THIS
15C SEARCH IS ALSO UNSUCCESSFUL, IT SEARCHES TO SEE IF IDN IS A TABLE C
16C OPERATOR DESCRIPTOR. IF IDN IS FOUND IN ANY OF THESE SEARCHES,
17C THIS SUBROUTINE RETURNS THE CORRESPONDING MNEMONIC AND OTHER
18C INFORMATION FROM WITHIN EITHER THE INTERNAL ARRAYS FOR REPLICATION,
19C REPLICATION FACTOR, TABLE D OR TABLE B DESCRIPTORS; OR FROM THE
20C KNOWN VALUES FOR TABLE C DESCRIPTORS. IF IDN IS NOT FOUND, IT
21C RETURNS WITH IRET=0. THIS SUBROUTINE IS IDENTICAL TO BUFR ARCHIVE
22C LIBRARY SUBROUTINE NUMTBD EXCEPT NUMTBD SEARCHS FOR IDN WITHIN ONLY
23C THE INTERNAL TABLE D AND B ARRAYS.
24C
25C PROGRAM HISTORY LOG:
26C 1994-01-06 J. WOOLLEN -- ORIGINAL AUTHOR
27C 1995-06-28 J. WOOLLEN -- INCREASED THE SIZE OF INTERNAL BUFR TABLE
28C ARRAYS IN ORDER TO HANDLE BIGGER FILES
29C 1999-11-18 J. WOOLLEN -- THE NUMBER OF BUFR FILES WHICH CAN BE
30C OPENED AT ONE TIME INCREASED FROM 10 TO 32
31C (NECESSARY IN ORDER TO PROCESS MULTIPLE
32C BUFR FILES UNDER THE MPI)
33C 2000-09-19 J. WOOLLEN -- ADDED CAPABILITY TO ENCODE AND DECODE DATA
34C USING THE OPERATOR DESCRIPTORS (BUFR TABLE
35C C) FOR CHANGING WIDTH AND CHANGING SCALE

54C TAB - CHARACTER*1: TYPE OF FXY VALUE THAT IS BIT-WISE
55C REPRESENTED BY IDN:
56C 'B' = BUFR Table B descriptor
57C 'C' = BUFR Table C descriptor
58C 'D' = BUFR Table D descriptor
59C 'R' = BUFR replication descriptor
60C 'F' = BUFR replication factor descriptor
61C IRET - INTEGER: RETURN VALUE (SEE REMARKS)

64C REMARKS:
65C THE INTERPRETATION OF THE RETURN VALUE IRET DEPENDS UPON THE
66C RETURN VALUE OF TAB AND THE INPUT VALUE IDN, AS FOLLOWS:
67C
68C IF ( TAB = 'B' ) THEN
69C IRET = positional index of IDN within internal BUFR Table B
70C array
71C ELSE IF ( TAB = 'C') THEN
72C IRET = the X portion of the FXY value that is bit-wise represented
73C by IDN
74C ELSE IF ( TAB = 'D') THEN
75C IRET = positional index of IDN within internal BUFR Table D
76C array
77C ELSE IF ( TAB = 'R') THEN
78C IF ( the F portion of the FXY value that is bit-wise
79C represented by IDN ) = 1 THEN
80C ---> regular (non-delayed) replication
81C IRET = ( -1 * ( the Y portion of the FXY value that is bit-
82C wise represented by IDN ) )
83C ---> where Y = the number of F=1 regular replications
84C ELSE
85C ---> delayed replication
86C IRET = positional index, I, of IDN within internal
87C replication array IDNR(I,1), where I=2,5
88C ---> IRET = I = 2 --> 16-bit delayed replication descriptor
89C ---> IRET = I = 3 --> 8-bit delayed replication descriptor
90C ---> IRET = I = 4 --> 8-bit delayed replication descriptor
91C (stack)
92C ---> IRET = I = 5 --> 1-bit delayed replication descriptor
93C END IF
94C ELSE IF ( TAB = 'F') THEN
95C IRET = positional index, I, of IDN within internal replication
96C array IDNR(I,2), where I=2,5
97C ---> IRET = I = 2 --> 16-bit replication factor descriptor
98C ---> IRET = I = 3 --> 8-bit replication factor descriptor
99C ---> IRET = I = 4 --> 8-bit replication factor descriptor
100C (stack)
101C ---> IRET = I = 5 --> 1-bit replication factor descriptor
102C ELSE IF ( IRET = 0 ) THEN
103C IDN was not found in internal BUFR Table B or D, nor does it

106C END IF
