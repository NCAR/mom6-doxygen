5C SUBPROGRAM: UFBSEQ
6C PRGMMR: WOOLLEN ORG: NP20 DATE: 2000-09-19
7C
8C ABSTRACT: THIS SUBROUTINE WRITES OR READS SPECIFIED VALUES TO OR FROM
9C THE CURRENT BUFR DATA SUBSET WITHIN INTERNAL ARRAYS, WITH THE
10C DIRECTION OF THE DATA TRANSFER DETERMINED BY THE CONTEXT OF
11C ABS(LUNIN) {I.E., IF ABS(LUNIN) POINTS TO A BUFR FILE THAT IS OPEN
12C FOR INPUT, THEN DATA VALUES ARE READ FROM THE INTERNAL DATA SUBSET;
13C OTHERWISE, DATA VALUES ARE WRITTEN TO THE INTERNAL DATA SUBSET}.
14C THE DATA VALUES CORRESPOND TO A SEQUENCE OF TABLE B MNEMONICS WHICH
15C ARE REPRESENTED BY A SINGLE TABLE A OR TABLE D SEQUENCE MNEMONIC.
16C THIS SEQUENCE MNEMONIC MAY ITSELF CONTAIN ONE OR MORE TABLE D
17C SEQUENCE MNEMONICS ALONG WITH TABLE B MNEMONICS, THE SEQUENCE
18C MNEMONICS HERE CAN USE EITHER DELAYED REPLICATION, REGULAR (I.E.,
19C NON-DELAYED) REPLICATION OR THEY CAN HAVE NO REPLICATION AT ALL.
20C HOWEVER, IN CASES WHERE THIS SUBROUTINE IS WRITING DATA VALUES TO
21C SEQUENCES USING DELAYED-REPLICATION, THE APPLICATION PROGRAM MUST
22C FIRST CALL BUFR ARCHIVE LIBRARY ROUTINE UFBINT TO PRE-ALLOCATE THE
23C SPACE NEEDED TO EXPAND THE DELAYED-REPLICATION SEQUENCE (THE NUMBER
24C OF REPLICATIONS IN DELAYED-REPLICATION IS SET TO ZERO BY DEFAULT).
25C (SEE BUFR ARCHIVE LIBRARY UFBINT DOCBLOCK REMARKS FOR MORE
26C INFORMATION.) IF UFBSEQ IS READING VALUES, THEN EITHER BUFR ARCHIVE
27C LIBRARY SUBROUTINE READSB OR READNS MUST HAVE BEEN PREVIOUSLY
28C CALLED TO READ THE SUBSET FROM UNIT ABS(LUNIN) INTO INTERNAL
29C MEMORY. IF IT IS WRITING VALUES, THEN EITHER BUFR ARCHIVE LIBRARY
30C SUBROUTINE OPENMG OR OPENMB MUST HAVE BEEN PREVIOUSLY CALLED TO
31C OPEN AND INITIALIZE A BUFR MESSAGE WITHIN MEMORY FOR THIS
32C ABS(LUNIN).
