program radiance_obs_seq_to_netcdf
 converts an observation sequence file to a netCDF file but only retains basic metadata for most observation types. Radiance observations, by necessity, have their metadata read in order to create temporary observation types that differentiate the channels and platforms, etc. Unique metadata to all other observation types is not converted. For example, the location of the radar and direction are not converted, but the location of the observation IS converted.
