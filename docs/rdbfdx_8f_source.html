6C PRGMMR: WOOLLEN ORG: NP20 DATE: 1994-01-06

8C ABSTRACT: THIS SUBROUTINE READS BUFR TABLE (DICTIONARY) MESSAGES FROM
9C AN INPUT BUFR FILE AND COPIES THEM INTO INTERNAL MEMORY (ARRAYS IN
10C COMMON BLOCK /TABABD/). IT IS ASSUMED THERE IS AT LEAST ONE
11C DICTIONARY MESSAGE AT THE BEGINNING OF THE FILE. THIS SUBROUTINE
12C PERFORMS A FUNCTION SIMILAR TO BUFR ARCHIVE LIBRARY SUBROUTINE
13C RDUSDX, EXECPT THAT RDUSDX READS FROM AN EXTERNAL FILE CONTAINING A
14C USER-SUPPLIED BUFR DICTIONARY TABLE IN CHARACTER FORMAT. SEE
15C DOCBLOCK IN RDUSDX FOR A DESCRIPTION OF THE ARRAYS THAT ARE FILLED
16C IN COMMON BLOCK /TABABD/.
