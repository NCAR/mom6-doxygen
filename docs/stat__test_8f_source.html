14C ABSTRACT: the STAT() function returns an array of system-dependent
15C information. one of the entries in the array is the number of bytes
16C in a file. the problem is that the index for this entry is different on
17C different systems. this program makes the STAT() call and prints out
18C the array contents. one of the numbers will match the size of a data
19C file and that will be the index required by the grabbufr program
20C (also in this directory). if grabbufr does not work, run this program
21C and verify the STAT index is correct.
