This routine is given information from an inflate type, scalar values for inflate and inflate_sd, the ensemble prior_mean and prior_var for an observation, the ensemble (or group) size, the observed value, observational error variance, and gamma_corr (see below for description). It computes updated values for the inflate and inflate_sd items using algorithms described in filter.html (including references to papers).

The gamma_corr parameter gives the localized prior correlation times the localization which is computed in the assim_tools routine filter_assim. For single state space inflation it is 1.0.
