subroutine model_mod_check::setup_run_array
private

configure the tests to run

if they used the existing 'test1thru', use that to select tests 1-N. otherwise: run_tests(:) is initialized to all -1. if the user didn't set test1thru and if run_tests() is still all -1, turn everything on. otherwise if the first entry isn't -1, they did specify something in that namelist entry and we turn on just those tests.
